# HR Interview Question Generator

An AI-powered web application that generates role-specific interview questions using Groq. Perfect for HR professionals, hiring managers, and interviewers looking to prepare thoughtful, relevant questions for any job position.

## Features

- **Job Title Input**: Enter any job title to generate contextual interview questions
- **AI-Powered Questions**: Groq openai model for intelligent, role-specific questions
- **Real-time Feedback**: Loading states and error handling for seamless user experience
- **Modern UI**: Clean, responsive design built with Tailwind CSS
- **Fast & Lightweight**: Built with Vite for blazing-fast development and production builds

## Tech Stack

- **Frontend Framework**: React 18 + TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS + Tailwind Vite plugin
- **AI API**: Groq - llama-3.3-70b-versatile
